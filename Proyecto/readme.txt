Instrucciones de Uso:

La carpeta Proyecto contiene todos los archivos y carpetas de un proyecto de Eclipse en un Workspace del mismo.
La carpeta WAR contiene un archivo de tipo .war para que se pueda importar en eclipse y sea más comodo su uso.


Configuraciones en Eclipse:

Una vez tenga el proyecto, se va a Run > Run Configurations > Apache Tomcat > Arguments > Working Directory > Other > Workspace > Nombre del proyecto importado > Ok > Apply > Close.

Cerrar Eclipse y volverlo a abrir para que los cambios se efectuen.


Para obtener un highScore.json de ejemplo:

Obtener el highScores.json de Proyecto > DataBase > highScores.json y moverlo a la carpeta DataBase (esta se crea automaticamente al correr el proyecto y crear una sesión).
